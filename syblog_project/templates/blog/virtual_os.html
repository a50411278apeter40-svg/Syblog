📖 사용 방법
1️⃣ ISO 업로드
리눅스, DOS, 기타 OS의 ISO 파일을 업로드하면 가상 CD-ROM으로 부팅합니다.
2️⃣ CPU 선택
20종 이상의 CPU 중 원하는 것을 선택. Intel, AMD, 레거시 CPU 모두 지원.
3️⃣ 상태 저장
가상 OS 실행 중 언제든 상태를 저장하고, 나중에 그 시점부터 다시 시작합니다.
4️⃣ 가상 HDD
동적 할당 방식으로 실제 용량만 사용. 최대 수십 GB 설정 가능.
